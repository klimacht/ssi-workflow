# Dewetting map gallery

PyMOL renderings of all maps in [`../`](..) — six benchmark proteins from
[Sinha et al. 2023][paper] plus PhLP5. Each protein has six images:
two map types × three orthogonal views.

For every protein, the **single-window η-map at φ\*** (left) is shown
next to the **per-atom φ\*_i map** built from all 13 biased windows
(right). Color scale in both: **red = hydrophobic, blue = hydrophilic**.

[paper]: https://doi.org/10.1021/acs.jpcb.3c04902
